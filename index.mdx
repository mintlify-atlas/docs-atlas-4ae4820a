Weird ERC20 is a curated collection of minimal Solidity implementations that demonstrate non-standard, dangerous, or surprising ERC20 token behaviors. Every token in this library is based on a real token deployed in the wild — many of which have been exploited to drain millions of dollars from DeFi protocols.

Smart contract developers and security auditors can use this library as a testing resource to validate their systems handle edge cases correctly, and as a reference when reviewing token integration code.

## Why this exists

The ERC20 "specification" is loosely defined and amounts to little more than an interface declaration. Even the few semantic requirements that exist are routinely violated by token developers. This creates serious challenges for smart contract developers who must interact with arbitrary ERC20 tokens.
